# Google Keep to Joplin

Plugin to import Google Keep notes to Joplin.

To get the Google Keep Notes, go to https://takeout.google.com/settings/takeout, deselect all but Keep, download the zip when ready then unzip it.

The plugin is accessible in Tools -> Import Google Keep Notes.
Select it and import the json files of the Google Keep Notes exported via Google Takeout you are interested in.
